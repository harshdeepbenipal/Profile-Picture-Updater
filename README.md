# Profile-Picture-Updater
EECS 4443 LAB 4 -  Media and Device Integration

## Architecture :
This application follows the Model–View–Controller (MVC) architecture pattern to separate the user interface from application logic. The View consists of the XML layout that displays the app title, buttons for capturing or selecting a photo, and an ImageView used to preview the updated profile picture. The Controller is implemented in the main Activity, which handles button clicks, runtime permission checks, and launching Intents for the camera and gallery. The Model represents the image data selected or captured by the user, which is then displayed as the updated profile picture in the interface.
